# 狒狒 World Cup

狒狒 World Cup 是一个面向世界杯竞猜的本地量化决策辅助工具。它通过正期望筛选、半 Kelly 注量控制、edge 校准与对冲套现计算，帮助使用者在竞猜中做出更理性的决策。

本项目为单文件 HTML 工具，纯前端、无服务器、不联网，所有数据仅存储于本地浏览器，不保存任何账号信息，也不会代替使用者进行任何投注。

## 如何使用

直接用浏览器打开 `ff_wc_betting_tool.html` 即可，无需安装任何依赖。

部分浏览器在 `file://` 协议下会禁用本地存储（localStorage），导致数据无法保存。如遇此情况，可在文件所在目录启动本地服务后访问：

```
python -m http.server 8000
```

随后在浏览器打开：

```
http://localhost:8000/ff_wc_betting_tool.html
```

建议定期通过「存档 → 导出」备份数据，以防清除浏览器缓存时丢失。

## 功能说明

- 总览：
  * 全局快照：总资产、目标进度、剩余天数、当前正期望机会数
  * 当前正期望机会：自动列出未结算、未过截止时间的正期望选项及其 EV 与建议注额
  * 目标进度：录入资金与日期，自动计算剩余可投注天数
  * EDGE 仪表：基于结算记录估计真实 edge，含置信区间与校准曲线
- 比赛：
  * 策略配置：策略目标、每日入账、Kelly 比例、单注上限、概率源权重、博彩赔率格式
  * 概率数据源：Opta、Dimers、Oddschecker 等快捷查询链接
  * 比赛表格：录入网站赔率、AI 概率、博彩赔率，自动计算综合概率、EV 与建议注额
  * 截止下注时间（北京时间）：过期后该场不再计入正期望机会
  * 赛后结算：自动扣减竞猜币、增加兑奖币并写入账本
- 套现：
  * 套现可行性分析：评估剩余场次与单场上限是否足以将竞猜币及时转化为兑奖币
  * 对冲套现计算器：按赔率倒数比例对三个结果分注，锁定确定性收益
- 记录：
  * 汇总：总下注、总回收、总盈亏、命中率、实测 edge
  * 明细：每笔结算的比赛、选项、注额、赔率、预测概率、赛果与盈亏
- 存档：
  * 多账号管理，各账号拥有独立的资金、比赛与账本
  * 支持导出完整存档（含资金，供个人备份）或仅比赛数据（仅含赔率与概率，可对外分享）
