套现可行性 · 竞猜币 → 兑奖币

最终仅兑奖币有效, 而兑奖币须通过投注获得, 且单场设有上限。当竞猜币积累较多, 而剩余场次 × 上限不足时, 上限将构成瓶颈——竞猜币无法及时完成转化。以下用于评估该瓶颈。

对冲套现计算器 · 稳定转化

对一场比赛的三个结果按赔率倒数比例分注, 使任一结果发生时回收金额相近, 从而锁定确定性收益。代价是确定承担约13%抽水, 换取100%的套现确定性。适用于优先保证资金转化、不愿承担单边落空风险的场景。注额按100取整, 故各结果实际回收略有差异。

使用说明 · 完整指南

工具定位

辅助世界杯竞猜的量化决策。竞猜含约 13% 抽水(赔率倒数和≈1.13), 故任一选项的默认期望为负(投1平均回收0.886)。本工具不保证盈利, 其功能为: ① 仅当用户估计的真实概率高于赔率隐含概率(正期望)时给出投注建议; ② 以半Kelly控制注量; ③ 记录战绩并估计真实edge; ④ 在需要时提供稳定套现的对冲方案。

快速上手(5步)
  1. 在 比赛 页配置顶部的策略目标、每日入账、博彩赔率格式。
  2. 点击 + 添加比赛, 录入竞猜平台的三项网站赔率(主胜/平/客胜)。
  3. 通过 比赛 页的概率数据源卡片(Opta/Dimers/Oddschecker)查询概率, 将胜率% 录入 AI概率源 栏, 博彩赔率录入 博彩 栏。
  4. 系统自动计算综合概率与期望EV, 标记正期望项并给出建议注额(已按100取整)。将实际投注额录入实际投入栏。
  5. 赛后于比赛卡片底部选择获胜方 → 结算并更新资产。盈亏录入 记录, 用于估计真实edge。
各栏目功能详解
总览
  • 全局快照条: 总资产、目标进度、剩余天数、当前正期望机会数。
  • 当前正期望机会: 自动列出所有未结算、未过截止时间的正期望选项及其EV与建议注额, 可经链接跳转至比赛页。
  • 目标进度: 录入当前兑奖币/目标/竞猜币, 及当前日期与活动结束日期(剩余天数自动计算 = 结束日−1天 − 当前日)。
  • EDGE 仪表: 依据结算记录估计真实edge, 含置信区间(区间跨越1时表明优势尚不显著, 应谨慎投注)与校准曲线(评估概率判断的准确性)。需至少20条记录。
比赛
  • 策略目标: "达到目标金额"(达标后停止)或"最大化最终收益"(纯Kelly增长)。
  • 每日入账: 200 / 300 / 无 / 自定义, 影响可投入资源的估算。
  • Kelly比例: 注量激进程度。0.5=半Kelly(推荐, 波动较低); 数值越高越激进, 不建议>1。
  • 单注上限: 单场最大注额(竞猜规则通常为1000)。源加权: 为AI1/AI2/博彩三类概率源设置不同权重。
  • 博彩赔率格式: 分数(8/11)/小数(1.73)/美式(-150), 选择数据源对应格式; 分数模式亦可直接录入小数。
  • 概率数据源: 四个快捷链接(Opta/Dimers/Oddschecker/莫古看球), 点击于新标签页打开查询。
  • 比赛表格: 录入网站赔率、AI概率%、博彩赔率(自动去抽水转概率)→ 输出综合P、EV、建议注额。每场可设截止下注时间(过期后不再计入总览机会)。支持▲▼调整顺序、撤回误删(最多5次)。
  • 实际投入: 录入实际注额(点击↤可采用建议额)。底部资产条实时计算剩余竞猜币与总资产。
  • 结算: 选择获胜方 → 结算并更新资产, 自动扣减竞猜币、增加兑奖币、写入账本。
套现
  • 套现可行性分析: 最终仅兑奖币有效, 且单场设有上限。此功能依据"剩余场次×上限×edge"评估竞猜币能否及时转化为兑奖币, 揭示上限是否构成瓶颈。场次由"剩余天数×每日场次"自动计算, edge自动取实测值。提供"维持上限/后期放开"两种情景, 以及"仅预警/影响下注"模式(后者在套现紧迫时自动提高建议注额)。
  • 对冲套现计算器: 按赔率倒数比例对三个结果分注, 使任一结果发生时回收相近, 以确定承担13%抽水换取100%套现确定性。支持"按投入"或"按拿回"两种计算方向。适用于优先保证资金稳定转化的场景。
记录
  • 汇总条: 总下注、总回收、总盈亏、命中率、实测edge。
  • 明细表: 每笔结算的比赛/选项/注额/赔率/预测概率/赛果/盈亏。该数据驱动总览的EDGE估计。
存档/切换号(右上角, 全局)
  • 管理多个账号, 各账号拥有独立的资金、比赛与账本。切换时自动保存当前账号。
  • 导出: ⬇完整存档(含资金, 供个人备份) / ⚑仅比赛数据(仅含赔率与概率, 可对外分享, 不含资金信息)。导入"仅比赛数据"时可选择载入当前账号(保留资金)或新建账号。
核心概念

抽水(margin): 三个选项赔率倒数之和≈1.13, 超出部分(约13%)为竞猜方抽成, 使所有选项默认呈负期望。

期望EV = 综合概率 × 网站赔率。EV>1 方具投注价值(正期望), 表明该选项被低估。

edge = 每投入1单位平均回收的兑奖币。>1为盈利, <1为亏损, ≈0.886为无优势(经抽水后)水平。该指标衡量识别"被低估选项"的实际能力, 小样本下不可靠, 须参考置信区间。

分数赔率换算: 小数赔率 = 分子÷分母 + 1。例: 8/11 → 8÷11+1 = 1.73; 14/5 → 14÷5+1 = 3.80。

策略要点: edge<1时不宜投注热门(抽水侵蚀显著), 若投注应选择高赔率冷门; edge≥1.15(且置信区间下界稳固)时, 宜投注稳健的正期望项。
